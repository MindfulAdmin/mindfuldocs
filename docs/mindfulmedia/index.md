# MindfulMedia

**Version:** 2.13.0

MindfulMedia is a comprehensive WordPress media management plugin with YouTube-inspired styling. It provides everything you need to create a professional video library, course platform, or media-rich website.

![MindfulMedia Browse Page](../assets/images/mindfulmedia/browse-page-full.png)

---

## What You Can Build

- **Video Libraries** - Organize videos by teachers, topics, and categories
- **Course Platforms** - Create playlists with modules and track progress
- **Podcast Sites** - Beautiful audio player with episode management
- **Membership Sites** - Gate content with MemberPress integration
- **Media Portals** - Netflix-style browsing experience

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Custom Post Type** | Dedicated `mindful_media` post type with custom fields |
| **Taxonomies** | Teachers, Topics, Categories, Playlists, Media Types |
| **Modal Player** | YouTube-style modal with playlist sidebar |
| **User Engagement** | Likes, comments, subscriptions, watch history |
| **My Library** | Personal library with continue watching |
| **MemberPress** | Gate content by membership level |
| **Email Notifications** | Notify subscribers of new content |
| **Page Builders** | Gutenberg blocks and Elementor widgets |

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- MemberPress (optional, for membership gating)
- WooCommerce (optional, for My Account integration)
